\chapter{绪论}

\section{研究背景与问题提出}

近年来，随着互联网金融信息的爆炸式增长，投资者的决策依据已经不再局限于价格、成交量、财务报表等结构化数据。新闻快讯、投资论坛讨论、机构研报以及搜索引擎中的热点信息，正在以前所未有的速度影响市场预期的形成与扩散。与此同时，大语言模型在自然语言理解、知识整合和复杂推理方面表现出显著能力，金融领域也由此开始从传统文本分类与情绪识别，逐步转向以金融专用大模型和金融智能体为核心的新阶段。BloombergGPT 展示了大规模金融语料训练金融专用模型的可行性，FinGPT 则进一步强调了开放数据管线与轻量微调在金融场景中的现实价值；多篇综述也指出，金融 LLM 已成为金融智能化研究的重要方向，但其实时性、幻觉、可解释性和评估体系仍然是核心挑战\citep{lee2024finllms,li2023llmfinance,nie2024llmfinancialapplications,yang2023fingpt,wu2023bloomberggpt}。

股票市场本质上是一个高度受预期驱动的复杂系统。价格变化并不只反映企业基本面，也反映了投资者对未来事件的理解、情绪与博弈结果。相比于结构化行情数据，文本信息往往更早承载市场预期变化。例如，论坛评论能够反映散户讨论热度与短期情绪，新闻能够传递突发事件与公开信息，研报则体现相对专业的分析框架和逻辑链条。因此，如何从多源异构文本中提取对未来走势真正有效的信息，并将其转化为可用于预测与决策的信号，具有明确的理论意义与应用价值。已有研究表明，社交媒体情绪与市场波动之间存在可观测关系，而大语言模型也已经表现出从新闻文本中识别未来市场反应的潜力\citep{bollen2011twitter,lopezlira2023chatgptstock}。

现有研究虽然已经分别在金融情绪分析、检索增强生成、社交媒体预测、金融大模型和强化学习交易等方面取得了重要进展，但仍存在若干不足。首先，很多研究仍以单一信息源为主，例如只使用新闻、只使用社交媒体或只做财报文本分析，缺少面向真实投研场景的多源异构信息统一处理框架。其次，不少研究建立在静态离线数据集之上，对“预测时实时获取最新信息”这一关键问题关注不足，而这恰恰是金融应用区别于一般 NLP 任务的重要特征。再次，社交媒体和论坛信息噪声很大，现有方法要么依赖手工构造情绪指标，要么依赖较复杂的结构模型，尚缺少一种兼顾灵活性、可解释性与工程可落地性的统一归纳方式。最后，尽管金融 LLM 研究快速增长，但综述普遍指出该领域仍面临幻觉、可追溯性、评估标准不统一等问题\citep{lewis2020rag,ye2024r2ag,lee2024finllms,li2023llmfinance,nie2024llmfinancialapplications,zhou2025expertopinion}。

基于以上背景，本文尝试从“真实预测流程”出发，而不是仅从静态数据集出发，构建一个能够在预测时实时接入外部信息的大语言模型框架。该框架不是通过人工设计复杂的情绪共识公式来计算板块或个股信号，而是通过大语言模型对多源文本进行结构化总结、证据整合和方向判断，从而更贴近实际投研分析场景。这一研究思路既回应了金融市场对实时文本理解的需求，也为大语言模型在金融预测中的应用提供了更工程化、可扩展的实现路径。

\section{研究目标与研究问题}

本文的总体目标是围绕股票短期走势预测任务，设计并实现一个面向真实应用场景的多源信息增强框架。该框架在预测时能够获取近期 Comments、News、Reports 以及外部检索结果，并通过大语言模型完成信息筛选、结构化汇总、证据整合和未来走势方向判断。

围绕这一目标，本文关注以下研究问题：第一，如何将评论、新闻、研报和外部检索结果等多源异构信息组织为适合大语言模型理解的输入形式；第二，如何利用大语言模型从噪声较高的金融文本中提取市场共识、潜在分歧与利好利空因素；第三，外部信息注入和领域监督微调是否能够提升模型在未来 1 日、3 日和 7 日涨跌方向预测任务上的表现；第四，如何在监督微调基础上进一步引入强化学习，为后续从方向判断走向策略优化提供扩展路径。

\section{本文的主要工作与贡献}

围绕上述研究目标，本文首先构建面向股票方向预测的多源信息获取与组织流程。在真实使用场景中，系统在需要预测时实时抓取最近的论坛评论、相关新闻、机构研报以及通过 Web API 返回的相关补充信息。当前工程实现中，该 Web API 模块主要通过 SerpAPI 接入搜索引擎结果，用于补充公告、财报、股价异动和最新消息等公开网页信息，并对不同来源文本进行清洗、截断、排序和结构化组织。

其次，本文设计基于大语言模型的多源总结与预测机制。针对 Comments、News 和 Report 三个分支分别生成结构化总结，再通过统一提示模板将三类信息综合输入模型，使其输出未来 1 日、3 日和 7 日的涨跌判断及主要依据。与传统方法不同，本文不预先手工定义情绪共识公式，而是将观点归纳、一致性分析和冲突识别交由大语言模型完成。

再次，本文开展离线可行性验证实验。考虑到历史时点的外部信息难以完全实时回放，本文采用预测未来 1 日、3 日和 7 日涨跌方向的方式进行离线测试，重点比较微调前后模型、是否引入额外外部信息两类因素对预测结果的影响。根据目前实验结果，微调后的 Qwen2.5-7B-Instruct 在接入外部信息源时取得了最高 56\% 的预测成功率，表明该方法具备一定有效性。

本文的主要贡献可以概括为以下三个方面：其一，面向真实预测流程而非纯静态数据集，提出了一个支持实时接入 Comments、News、Reports 和外部检索结果的多源信息融合框架；其二，采用“大语言模型驱动的结构化归纳”替代传统手工情绪指标构造方法，通过统一 schema、分支总结与综合判断实现对市场共识与分歧的抽取；其三，通过“微调前后 + 有无外部信息”的对照实验，验证了领域微调和外部信息注入对股票方向预测的增益作用，并为后续从“信息理解—方向预测—策略决策”构建完整闭环预留强化学习扩展路径。

\section{论文结构安排}

本文共分为七章。第 1 章为绪论，介绍研究背景、问题提出、研究目标、主要工作与论文结构。第 2 章介绍相关技术与理论基础，包括股票短期走势预测任务、多源金融信息特点、大语言模型、Prompt 方法、监督微调与强化学习对齐方法。第 3 章设计面向股票预测的多源实时信息增强框架，说明系统整体流程、多源信息获取、预处理筛选、统一组织与预测输出设计。第 4 章介绍模型训练与优化方法，包括监督微调数据构建、训练设计以及强化学习优化思路。第 5 章介绍系统实现与在线部署方案，包括数据获取、Web API 调用、缓存存储、模型调用和结果解析。第 6 章给出实验设计与结果分析，比较不同模型、不同信息输入和不同预测周期下的效果。第 7 章总结全文工作，分析当前不足并展望后续研究方向。
