# A股重点板块股票数据

收录 7 大板块、35 家代表性上市公司从 **2023-01-01 至 2026-01-01** 的日度行情数据。

## 板块与公司

| 板块 | 公司 |
|------|------|
| 电力设备与新能源 | 宁德时代、亿纬锂能、阳光电源、隆基绿能、比亚迪 |
| 医药生物 | 恒瑞医药、药明康德、复星医药、迈瑞医疗、云南白药 |
| 银行 | 招商银行、工商银行、平安银行、建设银行、兴业银行 |
| 半导体与电子 | 中微公司、北方华创、华虹半导体、韦尔股份、兆易创新 |
| 食品饮料（白酒） | 贵州茅台、五粮液、泸州老窖、洋河股份、山西汾酒 |
| 汽车 | 上汽集团、长城汽车、吉利汽车、广汽集团、江淮汽车 |
| 非银金融（券商） | 中信证券、东方财富、国泰君安、华泰证券、广发证券 |

## 目录结构

```
A股重点板块/
└── <板块>/
    └── <公司(代码)>/
        └── <年份>/
            └── <月份>.txt   # 该月每个交易日的行情数据
```

每个 `.txt` 文件包含以下字段：日期、股票代码、开盘、收盘、最高、最低、成交量、成交额、振幅、涨跌幅、涨跌额、换手率。

## 数据来源

通过 [akshare](https://akshare.akfamily.xyz/) 获取，A 股使用 `stock_zh_a_hist`，港股（吉利汽车）使用 `stock_hk_hist`，复权方式为不复权。

## 使用方式

运行 `fetch_stocks.py` 可重新拉取或更新数据：

```bash
pip install akshare pandas
python fetch_stocks.py
```
